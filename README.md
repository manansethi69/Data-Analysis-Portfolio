# Data Analytics Projects

This repository showcases data analytics projects built with **Python**, **SQL**, **Excel**, **Power BI**, **Tableau**, and **Azure**.[web:31][web:62] The goal is to demonstrate how these tools can be combined to solve real business problems end‑to‑end.

## Tech Stack

- **Python** – pandas, NumPy, matplotlib/seaborn for data cleaning, analysis, automation, and webscraping
- **SQL** – querying, joins, aggregations, window functions  
- **Excel** – data cleaning, formulas, pivot tables, charts  
- **Power BI & Tableau** – interactive dashboards and data storytelling  
- **Azure** – Azure SQL Database and storage for data pipelines

## Projects

- **Retail Sales Analytics** – Python + SQL + Power BI + Azure  
  - End‑to‑end sales analysis from raw CSV to Azure SQL and interactive Power BI dashboard.

- **Finance KPI Dashboard** – SQL + Excel + Tableau  
  - Financial KPIs modeled in SQL, summarized in Excel, and visualized in a Tableau dashboard.

- **Marketing Campaign Analysis** – Python + SQL + Power BI  
  - Campaign performance analysis with Python (pandas) and SQL, reported in Power BI.

- **Crypto Data Automation** – Python + APIs + Excel  
  - Automated crypto price pulls via API, stored and analyzed with Python and Excel.
